A task management app is a tool that lets you create tasks that represent things you want to do. 
They allow you to create tasks, add them to lists, check tasks off as completed, and a range of other features.
Some examples of this type of appare Todoist, Microsoft ToDo, Asana, Things3, or Omnifocus.

## Task Management Application Requirements
1. Users can register with an email address and password.
2. Users can create and manage their own tasks.
3. Users can create lists (projects) to organize tasks.
4. Each task belongs to exactly one list.
5. Tasks can be marked as completed.
6. Tasks store creation and completion timestamps.
7. Tasks can have an optional due date.
8. Tasks can be configured as recurring with customizable intervals (e.g., every day, every X days, every week, every X weeks, every month, every X months).
9. Users can create tags and assign multiple tags to a task.
10. Users can create workspaces/teams and invite other users to collaborate on shared lists and tasks.
11. Tasks can be assigned to a specific workspace member.
